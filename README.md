# Pokémon K-Means Clustering

This project demonstrates Unsupervised Learning using the K-Means Clustering algorithm on the Pokémon dataset.

## Project Overview

The project groups Pokémon based on their statistics such as:

- Attack
- Defense

The clusters are visualized using Matplotlib, where:
- Each color represents a different cluster
- Triangle markers represent cluster centroids

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Technique

- Unsupervised Learning
- K-Means Clustering

## Visualization

The scatter plot shows:
- Pokémon distribution
- Cluster assignments
- Centroid positions
- Connections between points and centroids

## Dataset

Pokémon dataset in CSV format.

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook
2. Install required libraries
3. Run all cells

## Output Example

The model creates clusters of Pokémon based on their battle statistics and visualizes the centroids of each cluster.
